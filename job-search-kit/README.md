# Job Search Kit — conversion assets

The agent gets you volume; these get you **conversion**. Use them alongside applications.

| File | Use it for |
| ---- | ---------- |
| [`interview-prep.md`](interview-prep.md) | STAR stories from your real wins + TPM/PM question bank + "tell me about yourself" |
| [`outreach-templates.md`](outreach-templates.md) | Referral asks, recruiter/HM outreach, follow-ups, and the right H1B framing |
| [`linkedin-profile.md`](linkedin-profile.md) | Keyword-optimized headline + About so recruiters find you (inbound) |
| [`positioning-one-pager.md`](positioning-one-pager.md) | One-page value prop + 3 case studies + 30/60-sec pitches |

**Highest-leverage order:** (1) update LinkedIn today → (2) send referral/recruiter outreach
for your top targets → (3) rehearse the STAR stories so you're ready when they reply →
(4) keep the one-pager handy for every call.
